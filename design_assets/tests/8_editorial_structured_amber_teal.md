# Define Card Slide — Editorial Publishing Style (Structured, Amber + Teal)

## Design Philosophy
This slide should feel like a premium editorial layout from a design-forward publication. Think Kinfolk meets Harvard Business Review — sophisticated, considered, with each element earning its place through purpose rather than decoration.

The aesthetic is quiet confidence. Nothing shouts. Everything breathes. British English throughout, with a tone that respects the reader's intelligence whilst building their capability.

## Colour Palette
Base is warm oat (#FDFBF7) with depth and comfort. Premium uncoated stock quality.

Primary text is dark navy/charcoal (#1C2A33), soft for comfortable extended reading.

Primary accent is amber/warm gold (#F2A900), used for editorial punctuation. Apply to underlines beneath key headings, primary emphasis, the HMW template highlight, process timeline connectors.

Secondary accent is deep ocean teal (#005B6E) for supporting elements, diamond diagram outlines, process step circles, synthesis flow lines. Provides structural depth throughout.

## Typography System
This is where editorial craft shows.

Headings use a refined serif — something with character and warmth. The card title ("Define: Closing the First Diamond") sits at 42–48pt with generous letterspacing. Section headers are 22–26pt, weighted medium.

Body text is a humanist sans-serif at 15–17pt, sized for comfortable reading with line height at 1.6. Never cramped, never floating.

Supporting text and captions use a lighter weight at 12–13pt, creating clear hierarchy through weight and size together.

Numbers use tabular lining figures in the body font, ensuring vertical alignment in the process timeline.

## Compositional Structure
The layout uses asymmetry and intentional imbalance to create visual interest whilst maintaining readability.

**Header zone (top 18% of slide):**
Title "Define: Closing the First Diamond" sits left-aligned with a thin amber underline (2px, extending 60% of title width). Metadata floats top-right in small caps, 11pt, teal: "PROCESS CARD · DESIGN THINKING". Generous space above and below creates breathing room.

**Content zone (middle 64%, asymmetric two-column):**
Columns are 40:60 split for visual tension.

Left column (narrower, 40%): "Why It Matters" as a pulled quote style — larger body text (18pt) with increased leading: "Synthesis transforms chaos into clarity. The Define stage forces divergent research into a single, actionable problem statement. Without it, teams solve the wrong problem beautifully." Below, "The Double Diamond" as a simplified diagram: two diamond shapes side by side, the left diamond filled with teal at 15% opacity with a bold amber vertical line at its closing point (the "define" moment). The right diamond is a faint outline only. Labels: "Discover" and "Define" beneath the first diamond, "Develop" and "Deliver" beneath the second. The closing point is labelled "Problem Statement" in small caps, amber.

Right column (wider, 60%): "The HMW Framework" presented as an elegant template card. A warm amber background tint (8% opacity) with a 1pt teal border. Inside, the template reads: "How might we [action verb] for [specific user] so that [measurable outcome]?" with each bracketed section in medium weight amber text, the surrounding words in regular charcoal. Below the template: a concrete example in italic — "How might we help employees build trusted relationships with new technology?" — with a thin amber left-border rule (2px). Below, "The Formula" set in small caps teal, with the equation: [User] + [Need] + [Insight] rendered as three teal-outlined boxes connected by amber + signs, each box large enough to read clearly.

**Process zone (bottom 18%):**
Four-step synthesis timeline runs as an elegant horizontal sequence. Numbers in circles (teal stroke, no fill) connected by a single hairline rule in amber. Steps: 1. Cluster findings → 2. Identify patterns → 3. Form insights → 4. Craft HMW statements. Step labels positioned below in sentence case. Timeline sits 2/3 across slide width, leaving right third empty — intentional asymmetry.

Key Principles anchor bottom-left as a tight list with amber square bullets (4pt):
- Converge, don't compromise
- Name the real problem
- Frame for action, not analysis

## Editorial Craft Details
Subtle details that signal quality without demanding attention.

Use an 8-column grid with 16px gutters, but break it intentionally. Perfect alignment is corporate; considered asymmetry is editorial.

Generous margins: 72px left and right, 64px top and bottom. Content has room to breathe.

Visual hierarchy through scale relationships: 3:2 ratio between title and section heads, 7:5 between section heads and body.

The double diamond diagram and HMW template should feel hand-considered, not template-generated. The formula boxes should have slight rounded corners (4px radius) for warmth.

## What Makes This Editorial
Restraint in colour application — amber and teal appear as warm accents, never dominating. The warm oat base does most of the visual work.

Whitespace as active element, not leftover space. Each gap is intentional.

Typography-led hierarchy. Use size, weight, and spacing to create structure.

The HMW template card is the centrepiece — it's the one element that combines both accent colours and draws the eye. Everything else supports it.

## Content Treatment
The double diamond appears in the left column with clear labelling showing the "define" closure point. The HMW framework shows both the template structure and a concrete example. The formula renders as a visual equation with distinct boxes. The 4-step process maintains sequence integrity. Key Principles remain visible but don't compete for attention.

This is teaching through considered design — each visual choice supports comprehension by reducing cognitive load whilst maintaining engagement through editorial craft.

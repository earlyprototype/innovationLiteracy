# Prompt: Assumption Mapping — Title Card (Amber + Teal)

**Target Image Ratio:** 16:9 

**System Instruction:**
You are an expert Information Designer and Infographic specialist. Generate a Super-Resolution 4K (3840x2160) high-fidelity 16:9 infographic. This is a flat digital UI graphic, edge-to-edge. Use Nano Banana 2.5's brilliant text-rendering with absolute typographic precision and zero compression artifacts.

**Visual Concept:**
The hero visual is a flat vector map or board with assumptions plotted as pins. Some pins are marked "known" (in teal), but most are marked with question marks (in amber). The pins are connected by fine red-amber threads showing dependencies between assumptions — creating a web of uncertainty. The overall feeling is: a detective's evidence board, but for innovation work, rendered as a flat 2D illustration.

**Content to Display:**
- **Title:** Assumption Mapping (large, bold serif, positioned top-left with generous breathing room)
- **Subtitle:** Making the implicit testable. (humanist sans-serif, light weight, directly beneath)
- **Card type label:** METHODOLOGY CARD · INTERVENTIONS (small caps, top-right, teal)

This is a **title card** — the visual metaphor is the star. Text is minimal and positioned to frame the illustration, not compete with it. No bullet points. No frameworks. No matrices. Just the title, subtitle, and the visual.

**Visual Aesthetic & Palette:**
- **Background:** Warm oat (`#FDFBF7`).
- Primary Accent: Amber (`#F2A900`) for question-mark pins, thread highlights, title underline.
- Secondary Accent: Deep Ocean Teal (`#005B6E`) for "known" pins, structural lines, metadata label.
- Thread colour: Red-amber (`#D4763A`) for dependency threads connecting pins.
- Text Color: Charcoal (`#1C2A33`).

**SIGNATURE DETAIL — The Interrupted Rule:**
Every horizontal divider line or rule on the card has a deliberate 12mm gap at approximately one-third from the left. The line starts, pauses with a clean break, then resumes. This applies to all horizontal separators — beneath the title, between zones, and any framing lines. The gap is always in the same position (one-third from the left edge). It should feel like a purposeful pause, not a rendering error.

**SIGNATURE DETAIL — The Oversized Ghost Number:**
The number "2" is rendered at approximately 200pt in a refined serif typeface, at only 4% opacity, positioned in the bottom-right area of the card and bleeding off the edge. It should be barely visible — a watermark that rewards close attention. The number is in charcoal (#1C2A33) at 4% opacity so it appears as a faint impression on the warm oat background. It sits behind all other content and should never interfere with readability.

**STRICT REQUIREMENT:** Must be a flat 2D vector-style infographic. Text must be perfectly readable and ultra-sharp. No 3D objects. No photos. Just pure high-DPI digital information design. Lossless quality render.

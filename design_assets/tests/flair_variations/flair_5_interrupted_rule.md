# Prompt: Flair Test 5 — Interrupted Rule (Amber + Teal)

**Target Image Ratio:** 16:9 

**System Instruction:**
You are an expert Information Designer and Infographic specialist. Generate a Super-Resolution 4K (3840x2160) high-fidelity 16:9 infographic. This is a flat digital UI graphic, edge-to-edge. Use Nano Banana 2.5's brilliant text-rendering with absolute typographic precision and zero compression artifacts. 

**Visual Concept (CONSTANT):**
The visual is based on the "Chaos to Clarity" metaphor. On the left side, show a dense "wall" of scattered, translucent post-it notes and data points. These notes are pulled and distilled into a single, sharp, elegant horizontal line or bar on the right side that holds the final Problem Statement. This is a vector-style infographic, not a photograph.

**Content to Display (CONSTANT):**
- **Title:** Define
- **Subtitle:** Closing the first diamond.
- **Main Body:** Synthesis transforms chaos into clarity.
- **The Craft:** "How Might We..." statements.
  - *Example:* "HMW help employees build trusted relationships?"
- **The Formula:** [User] + [Need] + [Insight].

**Visual Aesthetic & Palette (VARIABLE):**
- **Background:** Warm oat (`#FDFBF7`).
- **Palette (Innovation Warm):** Optimistic, energetic 'tech-forward' tones.
  - Primary Accent: Amber (`#F2A900`) for the "Clarity" bar and highlights.
  - Secondary Accent: Deep Ocean Teal (`#005B6E`) for the structural lines and "Chaos" nodes.
  - Text Color: Dark Navy/Charcoal (`#1C2A33`).

**SIGNATURE DETAIL — The Interrupted Rule:**
Every horizontal divider line or rule on the card has a deliberate 12mm gap at approximately one-third from the left. The line starts, pauses with a clean break, then resumes. This applies to all horizontal separators — beneath the title, between zones, and any framing lines. The gap is always in the same position (one-third from the left edge). It should feel like a purposeful pause, not a rendering error. ENSURE NO FOOTER LINES ARE PRESENT

**STRICT REQUIREMENT:** Must be a flat 2D infographic. Text must be perfectly readable and ultra-sharp. No 3D objects. No photos. Just pure high-DPI digital information design. Lossless quality render.

# E03 — Tools: Key Distinctions

## Slide Content

### Heading
Tools vs Toolkits vs Interventions

### Body
Objects vs method guides vs instructions

### Visual Direction
Three-way comparison. Tools: physical/digital objects (Mural, pens, AI assistants). Toolkits: document-based method guides (belong in resources for their respective elements). Interventions: instructions on how to use tools ("draw in 5 minutes", "dig deeper"). AI assistants are tools — ChatGPT for ideation, NotebookLM for analysis — enablers within a broader toolkit, not solutions.

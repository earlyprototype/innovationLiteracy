# E01a — Process: Definition

## Slide Content

### Heading
What is Process?

### Body
Structured, repeatable, learnable steps that guide innovation work. Gate-driven, not time-bound. Non-linear—loops back when deeper understanding is needed.

### Visual Direction
Design Thinking five-phase cycle (Empathise, Define, Ideate, Prototype, Test) shown as iterative loop with arrows returning to earlier phases. Emphasis on gates between phases (must pass criteria to advance). Clean, architectural diagram style. Loops visible—showing return to earlier phases when learning demands it.

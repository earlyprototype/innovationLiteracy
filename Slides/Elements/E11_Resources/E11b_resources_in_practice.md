# E11 — Resources: In Practice

## Slide Content

### Heading
Resources You Required

### Body
Protected time. Prototyping materials. Facilitation capacity. Collaborative tools.

### Visual Direction
The workshop's own resource requirements made visible. Time protection for the process to work, prototyping materials, facilitation capability, collaborative tools (Mural). Resource allocation signals organisational priority — claiming to value innovation without allocating dedicated resources is cargo culting.

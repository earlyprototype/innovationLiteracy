# E11c — Resources: In Practice

## Slide Content

### Heading
The Five Layers of Value

### Body
Capability Value. Learning Value. Engagement Value. Pipeline Value. Strategic Positioning. 

### Visual Direction
Five-layered pyramid or concentric circle diagram. Each layer represents a different type of value generated by innovation work, demonstrating that ROI is not the only metric. Neon yellow (#DFFF00) highlights the less obvious, structural layers like Capability and Learning that form the foundation for Pipeline and Strategic Value.

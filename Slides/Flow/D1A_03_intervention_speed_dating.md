# D1A — Intervention Speed-Dating

## Slide Content

### Heading
Five Methods. Five Minutes Each.

### Body
Apply each one to the Wallet Project

### Visual Direction
Five distinct stations visually represented. Energetic, rotation-based feel. Each station has a clear identity:

1. How Might We Statements — reframe the problem
2. 2x2 Prioritisation Matrix — impact vs effort
3. Assumption Mapping — test what you assumed
4. Stakeholder Mapping — see the system
5. SWOT Analysis — strengths, weaknesses, opportunities, threats

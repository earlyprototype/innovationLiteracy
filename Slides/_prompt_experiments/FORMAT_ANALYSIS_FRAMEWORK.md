# Format Experiment Analysis Framework
**Slide Generation for Educational Workshop Material**

---

## 1. EDUCATIONAL EFFECTIVENESS CRITERIA

### 1.1 Content Accuracy
- [ ] Slide count matches source material (4 slide preps provided)
- [ ] Content from slide preps accurately represented
- [ ] No invented/fabricated content beyond source
- [ ] Technical terminology correctly rendered

### 1.2 Information Hierarchy
- [ ] Title card clearly identifies element position (Element 01 of 11)
- [ ] Definition slide presents core concept succinctly
- [ ] In Practice slide connects to workshop experience
- [ ] Distinctions slide clarifies boundaries

### 1.3 Visual Clarity for Learning
- [ ] Text legible at presentation scale
- [ ] Key information prominent and accessible
- [ ] Visual aids support comprehension (not decoration)
- [ ] Technical annotations clear and purposeful

### 1.4 Workshop Appropriateness
- [ ] Professional tone suitable for adult learners
- [ ] Technical drawing aesthetic appropriate for innovation content
- [ ] Consistent with framework pedagogy (structure, boundaries, application)

---

## 2. AESTHETIC QUALITY CRITERIA

### 2.1 Style Execution
- [ ] Tech/neon art aesthetic maintained consistently
- [ ] Blueprint/technical drawing metaphor executed
- [ ] Avant-garde positioning achieved without sacrificing clarity
- [ ] Monochrome + neon yellow colour discipline adhered to

### 2.2 Composition Quality
- [ ] Grid structure visible and purposeful
- [ ] Geometric shapes (circle, square, triangle, hexagon) properly deployed
- [ ] Photo cutout treatment (no rectangles)
- [ ] Leader lines and annotations technical drawing standard

### 2.3 Typographic Excellence
- [ ] Serif/sans-serif mixing creates hierarchy
- [ ] Text positioned on grid alignment
- [ ] Metadata typography (Element 01/11) appropriately scaled
- [ ] British English spelling maintained

### 2.4 Visual Cohesion
- [ ] Consistent background colour (#E0E0D0) across all slides
- [ ] Neon yellow (#DFFF00) accent usage disciplined
- [ ] Layer ordering maintained (grid → shapes → photos → text)
- [ ] Slides feel like parts of unified deck

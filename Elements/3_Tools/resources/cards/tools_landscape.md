# The AI Tools Landscape (2026)

## Categories

### 1. AI App Builders & Rapid Prototyping
Optimal for rapid, low-fidelity prototyping to test desirability before committing to engineering.

- **[Bolt.new](https://bolt.new/)** — Browser-based, WebContainers, zero install
- **[Lovable](https://lovable.dev/)** — Conversation-first, full-stack with one-click deploy
- **[v0 by Vercel](https://v0.dev/)** — Generative UI → production-ready React/Tailwind

### 2. AI-First IDEs
When prototyping moves to feasibility and architecture.

- **[Cursor](https://cursor.com/)** — VS Code fork, synthesis engine for research + docs
- **[Windsurf](https://www.windsurf.ai/)** — Deep codebase memory, MCP support

### 3. Visual Design & Multimodal Creation

- **[Figma AI](https://www.figma.com/)** — Collaborative interface design with AI features
- **[Playground](https://playground.com/)** — Precise prompt adherence, text-to-image
- **[Midjourney](https://www.midjourney.com/)** — Conceptual art, atmospheric visualisations
- **[Luma](https://lumalabs.ai/)** — Text, image, video, 3D generation agents
- **[Adobe Firefly](https://firefly.adobe.com/)** — Enterprise-safe generative image creation

### 4. Synthesis, Research & Knowledge Management

- **[NotebookLM](https://notebooklm.google.com/)** — Document synthesis, structured outputs from uploads
- **[Perplexity](https://www.perplexity.ai/)** — Verified, cited research
- **[Claude](https://claude.ai/)** — Long-document synthesis, complex coding
- **[ChatGPT](https://chatgpt.com/)** — Advanced reasoning, voice interactions

## Strategic Recommendations

- **Standardise on Markdown** — enables synthesis tools to parse the full knowledge base
- **Match tool to task** — App Builders for desirability, IDEs for engineering
- **Consider interoperability** — MCP enables AI agents to pull live data from external systems

# ⚡ QUICK REF: Facilitation Patterns

## Pattern 1: Assisted Synthesis (Discovery)
Human extract → AI patterns → Human interpret → Human decide
**Language:** "AI spotted patterns — which resonate with what you heard?"

## Pattern 2: Expanded Ideation (Ideate)
Human ideas first → AI expand → Narrow strategically → Curated evaluation
**Language:** "We start with human thinking, then AI expands"

## Pattern 3: Prompt Co-Creation (Prototype)
Pairs write prompt → Surface reasoning → Generate → Iterate as work
**Language:** "You're writing the prompt, not me"

## Pattern 4: Verification Protocol (Test)
AI synthesis → Four Gates group exercise → Human interpretation
**Language:** "AI identified themes — now let's verify them"

## Universal Principles
1. AI accelerates cognition, doesn't substitute
2. Verification is not optional
3. Failure is diagnostic ("This nonsense is useful")
4. Process integrity over speed
5. Co-creation over control

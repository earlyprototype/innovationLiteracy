# Quick Reference: Space Design Checklist

## Before a Session

- [ ] Remove boardroom seating arrangement
- [ ] Create standing-height work surfaces
- [ ] Cover walls in writable paper
- [ ] Place prototyping materials visibly (not locked away)
- [ ] Set up quiet area for individual processing
- [ ] Check lighting — adjustable if possible
- [ ] Test digital tools (Mural/Miro boards, projectors)

## Zone Check

| Need | Zone | Setup |
|------|------|-------|
| Team alignment | Collaborative Hub | Circle seating, shared screen |
| Divergent ideation | Ideation Zone | Standing, whiteboards, sticky notes |
| Building/making | Active Making | Materials, craft supplies, paper |
| Individual reflection | Retreat | Quiet corner, noise-cancelling options |

## Remote Equivalent

| Physical | Digital |
|----------|---------|
| Wall space | Mural/Miro board with templates |
| Sticky notes | Digital sticky notes |
| Standing/moving | Camera-off individual work |
| Quiet retreat | Breakout room / async time |

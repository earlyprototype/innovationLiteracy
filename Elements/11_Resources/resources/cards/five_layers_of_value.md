# Five Layers of Innovation Value

## Why It Matters

When innovation projects present value to decision-makers, they typically default to financial ROI — the language operations speaks. But early-stage innovation produces value that ROI cannot capture: learning, capability development, cultural shift, strategic optionality. This card provides a structured framework for articulating the full value stack.

## The Five Layers

### 1. Learning Value
**What you discovered** — insights about users, market, technology, organisational readiness.
- "We learned that 60% of users don't use the core feature we assumed was critical"
- Measurable: insights generated, assumptions tested, unknowns resolved

### 2. Capability Value
**What you built in people** — skills, mindsets, methods now embedded in team.
- Directly maps to WEF Future of Jobs competencies
- Measurable: new methods adopted, team confidence in innovation process

### 3. Option Value
**Strategic options you created** — ideas, prototypes, relationships now available for future use.
- Like financial options: cost little to create, potentially enormous upside if exercised
- Measurable: number of validated concepts, partnerships initiated

### 4. Cultural Value
**How the organisation changed** — permission structures, collaboration norms, failure tolerance.
- Invisible but crucial: teams now share work-in-progress, challenge constructively
- Measurable: behavioural changes observed, psychological safety indicators

### 5. Financial Value
**Direct economic impact** — revenue, cost savings, efficiency gains.
- Appropriate ONLY for mature-stage, validated innovations
- Measurable: standard financial metrics

## Audience-Specific Framing

| Audience | Lead With | Support With |
|----------|-----------|-------------|
| **Finance** | Layer 5 (Financial) + Layer 3 (Option Value as hedge) | Layers 1-2 |
| **HR/People** | Layer 2 (Capability) + Layer 4 (Culture) | WEF data |
| **Strategy** | Layer 3 (Options) + Layer 1 (Learning) | Competitive intelligence |
| **Board** | Layer 5 if available; otherwise Layer 3 | Risk reduction narrative |

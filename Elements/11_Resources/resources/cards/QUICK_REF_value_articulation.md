# Quick Reference: Value Articulation by Audience

## The Five Value Layers

| Layer | What It Is | When to Use |
|-------|-----------|-------------|
| 1. **Learning** | Insights, validated/invalidated assumptions | Always — foundation |
| 2. **Capability** | Skills, methods embedded in people | HR, workforce strategy |
| 3. **Option** | Strategic possibilities created | Strategy, board |
| 4. **Cultural** | Behavioural/organisational change | People, transformation |
| 5. **Financial** | Revenue, savings, efficiency | Finance, late-stage only |

## Audience Cheat Sheet

| Talking to... | Open with... | Avoid... |
|---------------|-------------|----------|
| **CFO** | Option value + financial projections | "We had great workshops" |
| **CHRO** | Capability + WEF skills alignment | Pure financial metrics |
| **CEO/Board** | Strategic options + risk reduction | Early-stage ROI claims |
| **Sponsor** | Learning velocity + next stage plan | Vanity metrics |

## Key Phrase

> "This £10k investment bought us 3 validated assumptions, 2 invalidated assumptions (saving £200k+ in wrong-direction build), and a team now fluent in structured testing methodology."
